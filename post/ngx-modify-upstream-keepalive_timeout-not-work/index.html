Ngx Modify Upstream Keepalive_timeout Not Work

2020-08-18 320 words 1 min read

Contents

  • Reference:

nginx upstream 修改Keepalive_timeout 不生效；

nginx 作为代理连接upstream上游时，默认使用http1.0, 不能建立ngx –> upstream的长连接；

需要修改为1.1协议：

  • ngx相关配置

upstream test1_bak {
	server 192.168.150.66:80;
	keepalive 10;
	keepalive_timeout 100s;
}

server {
	listen       8088;
	server_name  testabc.dyf.com;

	location / {
		proxy_pass http://test1_bak;
		proxy_http_version 1.1;
		keepalive_timeout 100s;
		proxy_set_header Connection "";
	}
}
  • upstream(Ngx)相关配置：

http {
	keepalive_timeout  75s;
}
